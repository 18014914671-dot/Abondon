-ADS---------------------------------------------------------------------------------------------------------------------------------

- Для того чтобы работала реклама в приложении нужно
активировать Unity Ads в Unity Services!

- Реклама в игре (Scripts/Ads): 
RewardedAdsMenu (Дополнительные монетки в меню) и RewardedAdsPlay (Двойные монетки после поражения)

- Как изменить "Двойные монеты"?
- Находяться они на сцене "Play" -> "Canvas" -> "DoubleCoinMenu"
- Изменить значение выпадения монет можно в скрипте "RewardedAdsPlay"

- Если у вас перестала работать реклама, выключите её перезапустите Unity и включите, или поищите ответ в Google!

-LANGUAGE----------------------------------------------------------------------------------------------------------------------------

- Как изменять или добавлять языки в игре?

Заходим в папку (Scripts/Language), в папке лежат два скрипта "MenuLanguage" (Текст в Главном меню) и "PlayLanguage" (Текст в игре)
В скрипте при первом запуске игры вам предлагаеться выбрать язык (Language = PlayerPrefs.GetString("Language", "Start");)
Для того чтобы добавить другой язык вам нужно скопировать два метода одного из языков (EnglishStart и English)
После чего изменяем ваш новый язык:
- Изменяем название языка
- Изменяем текст для всех параметров
- Изменяем строку if(Language == "Ru") на -> if(Language != "Eng") (Вместо "Eng" может быть ваш язык)
- Изменяем Language = "Eng"; на ваш новый язык

- В "void Start" добавляем ваш язык:
if(Language == "YourLanguage") {
	YourLanguage();
	YourLanguage.SetActive(true);
	RuChek.SetActive(false);
	EngChek.SetActive(false);
}

- Так же не забываем добавить кнопку нового языка в "GameSettings", "LanguageSelection" и добавить новый язык в скрипт "PlayLanguage"!
- Для изменения языка вам нужно просто переименовать один из языков на свой, изменить текст параметров и добавить картинку языка!

-SKINS-SHIPS-------------------------------------------------------------------------------------------------------------------------

- Как поменять скины кораблей?
- Скины кораблей находяться "Sprites/Ships"
- Prefabs кораблей находяться "Prefabs/Enemy" или "Prefabs/Player/Ships"

- Изменение скина кораблей врагов происходит с помощью префабов и скрипта "EnemyGenerator" ("GeneratorMES" на сцене)
- Изменение скина корабля игрока происходит в коде с синхронизацией выбранного корабля в главном меню "ShipController"

-MORE-OF-OUR-GAMES-------------------------------------------------------------------------------------------------------------------

- Как изменить кнопку "Больше наших игр"?
- Если вы просто хотите изменить ссылку то вставьте свою ссылку "MainCamera" -> "MainMenu" -> "URL" (void GooglePlayPage)
